Ganeti RPM Packaging for el7
============================

Ganeti RPM Packaging for RHEL/CentOS/Scientific Linux 7.

Packaging status
----------------

* RHEL/CentOS/Scientific Linux 7.x: 2.12.0

* `Other distribution version <https://github.com/jfut/ganeti-rpm/>`_

Build the package
-----------------

* Build all packages::

  ./package.sh -a

* Build the specified package(s) only::

  ./package.sh -p [PACAKGE]
